# Proceso de agrupacion de ciudades por clima y temperatura

## Funcionalidades

- Leer Datos: Carga datos de las ciudades desde un archivo JSON.
- Procesar Datos: Extrae y organiza la información sobre las ciudades, clima, temperatura y más.
- Agrupar Ciudades: Agrupa las ciudades en función de sus tipos de clima y rangos de temperatura.
- Guardar Resultados: Guarda la información procesada en un nuevo archivo JSON.
- Segundo Plano: El proceso esta configurado en segundo plano para que se repita cada 5 minutos

## Que se necesita?

- **.NET 8 SDK**: Puedes descargarlo [aquí](https://dotnet.microsoft.com/download).
- **Un archivo JSON con datos de clima**: Asegúrate de tener un archivo JSON con datos de clima similar al ejemplo que te proporcionamos.
